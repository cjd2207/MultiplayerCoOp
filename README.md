# MultiplayerCoOp

A demonstration project built with **Unreal Engine 5.6.1** showcasing core multiplayer functionality using a cooperative gameplay model.

# Project Overview

**MultiplayerCoOp** is a sample Unreal Engine 5 project designed to explore and demonstrate:

- Peer-to-peer and server-client networking models  
- Player spawning and replication  
- Movement and animation synchronization  
- Multiplayer user interfaces (lobby, ready states, etc.)  
- Blueprint and/or C++ networking features  
- Session management (join/host functionality)

This project serves as a learning tool or starter template for developers building multiplayer experiences in Unreal Engine 5.6.1.

---

# Technologies Used

 Unreal Engine **5.6.1**
 Blueprint & C++ (mixed usage)
 Unreal Networking (Replication, RPCs, Sessions)

# Setup Instructions
Clone the repository
git clone https://github.com/cjd2207/MultiplayerCoOp.git

Open the project
Double-click MultiplayerCoOp.uproject or open via Unreal Engine 5.6.1
Enable required plugins (if prompted):
Online Subsystem (e.g., Null or Steam)
Networking-related plugins
Build the project (if using C++):
Open in Visual Studio
Build the solution (Ctrl + Shift + B)
